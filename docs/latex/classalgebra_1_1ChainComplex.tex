\subsubsection*{template$<$class T$>$\newline
class algebra::\+\+Chain\+Complex$<$ T $>$}
Class representing a chain complex with coefficients {\ttfamily T}. 

Chain complex is a free module over T with submodules {\ttfamily (C\+\_\+d, ..., C\+\_\+2, C\+\_\+1, C\+\_\+0, C\+\_\+(-\/1))} and linear operators {\ttfamily B\+\_\+n :\+ C\+\_\+n -\/\texorpdfstring{$>$}{>} C\+\_\+(n-\/1)} satisfying B\+\_\+n\texorpdfstring{$\ast$}{*}\+B\+\_\+(n+1) == 0 (the chain complex condidition).
