\doxysubsection{Detailed Description}
An euclidean domain concept. 

{\ttfamily Euclidean\+Domain} concept declares that a type models a mathematical definition of an Euclidean domain, that is a commutative ring, where we can define an euclidean function f, that satisfies two conditions:\+
\begin{DoxyEnumerate}
\item For a and nonzero b there exist q and r satisfying a = q \texorpdfstring{$\ast$}{*} b + r and f(r) \texorpdfstring{$<$}{<} f(b)
\item For a and b nonzero we have f(a) \texorpdfstring{$<$}{<}= f(b)
\end{DoxyEnumerate}

Additionally, we require function {\ttfamily divide}, which returns the numbers q and r.\hypertarget{conceptalgebra_1_1EuclideanDomain_autotoc_md5}{}\doxysubsection{\texorpdfstring{Semantic requirements}{Semantic requirements}}\label{conceptalgebra_1_1EuclideanDomain_autotoc_md5}
